
# A command line tool to help patch your Konflux pipelines

## Usage

```bash
curl -sLO https://github.com/simonbaird/konflux-pipeline-patcher/raw/main/pipeline-patcher && chmod a+x ./pipeline-patcher
./pipeline-patcher help
```
